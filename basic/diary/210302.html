고정좌표

> position:fixed;
> relative - 상대좌표 / absolute - 절대표 / fixed - 고정좌표

이미지 요소 세로,가로 배치/인라인과 블록

> display 를 block 지정(display:block;) 후 margin, padding 이용하여 배치
> display 를 inline 지정(display:inline;) > 인라인: 크기 X, 상하마진 X ,패딩 O
> 블록: 크기 O, 상하마진 O ,패딩 O
> 문법: 인라인>블록X, 인라인>인라인 O, 블록>인라인O, 블록>블록X

제목 텍스트 대신 이미지로 대체

h1에 background:url(이미지경로) no-repeat;
-> + h1 span { display:none;} 작성하면 h1 글자는 웹페이지에 보이지않고 이미지만 보임(html파일에서는 보임)

목록 문단, 정의 문단을 테이블 형식으로 배치

> display:table;로 두고 그 안에 있는 dd태그나 li태그에 display:table-cell; 넣어줌
> border-spacing:3px; - 박스4개에 3px 간격 > border-collapse:collapse; - 두줄 -> 한줄

다단에 구분선 표시

column-count:3; - 다단 수
column-rule:1px dashed #666; - 다단 사이 선 디자인
column-gap:40px; - 다단 사이 간격

transform 속성으로 요소 변경

transform:rotate(25deg); - 각도
transform-origin:20% 40%; - 중심점
