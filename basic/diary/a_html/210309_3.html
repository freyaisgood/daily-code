Media Query

width, height 화면 해상도 너비와 높이
min-width, min-height 화면 해상도 최소 너비와 높이
max-width, max-height 화면 해상도 최대 너비와 높이
device-width, deviece-height 기기의 너비와 높이
min-device-width, min-deviece-height 기기의 최소 너비와 높이
max--device-width, max--deviece-height 기기의 최대 너비와 높이
device-aspect-ratio 기기 화면의 영상 비율
min-device-aspect-ratio, max-device-aspect-ratio 기기 화면 영상 비율의 최소와 최대
resolution 비트맵 미디어 타입의 해상도
min-resolution, max-resolution 비트맵 미디어 타입의 해상도의 최소와 최대
color 컬러의 비트
min-color, max-color 컬러의 비트의 최소와 최대 수
color-index 컬러의 순람표
min-color-index, max-color-index 컬러의 순람표의 최소와 최대 수
monochrom 흑백 프레임 버퍼의 픽셀당 비트 수
min-monochrom, max-monochrom 흑백 프레임 버퍼의 픽셀당 비트 최소와 최대 수
orientation 화면 회전시 사용하며 portrait, landscape 값에서 선택함
scan 미디어 타입이tv일 경우값에서 선택함
grid 비트맵 그리드의 사용 선택을 체크함
