CSS 적용하기

text-decoration:none; : 밑줄이 없어짐
a:hover{컬러;} : 마우스 커서를 대면 컬러바뀜
margin:auto; : 텍스트가 아닌 요소들을 중앙정렬
list-style:none; : li 태그 앞에 있는 기호 없어짐

class와 id

  1. clss와 id의 이름은 자유롭게 할 수 있지만, 띄어쓰기가 들어가면 안되고, 꼭 하나의 완전한 단어이어야함.
  2. 복합단어 일 경우 _,-,대문자로 표기.
  3.    ex) freya_is_good(스네이크케이스), freya-is-good(케밥케이스), FreyaIsGood(카멜케이스)
  4. 네이밍시 영어로 시작해야 하며 숫자, 일부특수기호(_,-,$)혼용 가능.
  5. id는 한문서에 한번만 사용
  6. class는 한문서에 다중사용
  7. style에서 class는 . 으로 시작, id는 # 으로 시작

속성 선택자 사용하기

a 요소를 이용하여 선택자 정의

  1. a[선택자]{css요소} 를 하게 되면 원하는 a요소 안 선택자에게만 css 적용가능
  2. a[선택자="선택자안의 원하는 텍스트"] : 일치하는 텍스트에게만 css 적용가능
  3. a[선택자^="선택자안의 원하는 텍스트"] : 그 텍스트로 시작하는 텍스트에게만 css 적용가능
  4. a[선택자$="선택자안의 원하는 텍스트"] : 그 텍스트로 끝나는 텍스트에게만 css 적용가능
  5. a[선택자*="선택자안의 원하는 텍스트"] : 그 텍스트가 포함되는 텍스트에게만 css 적용가능

자식 선택자 사용하기

ex) h2요소 안에 삽입된 span 요소 이용 선택자 정의

h2>span{원하는 css style 적용}

ex) dd요소 안에 삽입된 span 요소 이용 선택자 정의

dd>span{원하는 css style 적용}
