선택자 사용하기

인접 형제 선택자 사용하기

ex)li 요소와 인접해있는 다른 형제 li 요소를 이용하여 선택자를 정의

li+li{css}을 넣어주면 첫번째에 있는 li 제외 나머지에 스타일이 입혀짐.

링크 관련 가상 클래스 선택자

a 요소의 상태에 따른 선택자를 정의

  1. a:link{css} : 방문 전의 링크 선택
  2. a:visited{css} : 방문 후의 링크 선택
  3. a:hover{css} : 마우스를 올렸을 때
  4. a:achive{css} : 클릭 할 때

동적 가상 클래스 선택자

  1. input:hover{css} : input 요소의 상태에 따른 선택자를 정의(마우스 커서 가져다 대면 바뀜)
  2. input:focus{css} : 양식의 입력칸 등에 클릭이나 탭하는 상태(탭키로 이동할때 바뀜)

첫 번째, 짝수 번째, 세 번째 li 요소 꾸미기

li:first-child{css} 혹은 nth-child{css} : li요소 첫번째만

li:nth-child(even){css} : 짝수번째 li요소만

li:nth-child(odd){css} : 홀수번째 li요소만

li:nth-child(n){css} : n번째 li 요소만

순서와 관련있는 가상 선택자

p:first-letter{css} : p태그 첫번째 글자만

p:first-line{css} : p태그 첫번째 줄만

내용 추가와 관련있는 가상 클래스 선택자

h2:before{css} : 선택자 앞에 내용을 추가

h2:after{css} : 선택자 뒤에 내용을 추가

전체 선택자

*{css} : html 전체 요소의 CSS 속성을 정의

그룹 선택자

h1,h2,h3,h4{css} : 여러 선택자에 같은 CSS 속성을 정의

소속을 정하는 트리 구조의 하위 자손 선택자

#helthBox2 h2{css} : #helthBox2 하위에 있는 h2 태그 속성을 저의
