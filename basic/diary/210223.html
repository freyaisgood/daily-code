글자와 관련된 속성

font-family - 글씨체
em단위 - 상대단위
font-variant:small-caps; - 소문자를 대문자로
font 순서 - style variant weight size line-height family
ex)font:italic small-caps 30px "Times New Roman";

문단에 관련된 속성

word-spacing - 단어사이의 간격
letter-spacing - 글자간의 간격(자간)
line-height - 텍스트 줄 사이의 거리
text-transform:uppercase or lowercase - 대문자로 or 소문자
text-indent - 들여쓰기
text-decoration:overline - 글자 위 라인
text-decoration:underline - 글자 아래 라인

글 문단의 흐름 방향, 공백 처리에 관련된 속성

direction:rtl - 문단의 흐름방향 [rtl(right to left):오른쪽, ltr:왼쪽]
white-space:nowrap - 줄바꿈이 되지 않음
white-space:pre - 공백을 사용자가 입력한 만큼 표시.
줄바꿈X (사용자가 Enter로 직접 줄바꿈 한것 만 적용)
- pre태그 참고자료 -

목록 문단에 이미지, 위치, 스타일 제어하기

ex)
id 안에
list-style-image:url(이미지위치)
list-style-position:inside or outside
list-style-type:upper-글씨체 (lower-글씨체 소문자)

인용부호, 특정 문자열과 같은 콘텐츠 생성하기

margin 적용 순서
margin-top margin-right margin-bottom margin-left
ex)
h1:before{ content:open-quote; }
h1:after{ content:close-quote; }
위 처럼 적용하면 h1글자 앞(before), 뒤(after) "(open-quote)"(close-quote)적용됨
